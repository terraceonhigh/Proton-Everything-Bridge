{{.Name}}'s Bridge Login

Run these commands on the server to authenticate {{.Name}}'s Proton account with each bridge.

1 Mail Bridge

docker compose -p user-{{.Name}} \ -f docker-compose.user.yml \ exec proton-mail-bridge \ protonmail-bridge --cli > login > (enter Proton credentials) > info (note the bridge password) > exit

2 Calendar Bridge

docker compose -p user-{{.Name}} \ -f docker-compose.user.yml \ exec proton-calendar-bridge \ proton-calendar-bridge --login

3 Drive (rclone)

docker compose -p user-{{.Name}} \ -f docker-compose.user.yml \ exec rclone-webdav rclone config > n (new remote) > proton (name it 'proton') > protondrive (type) > (follow prompts for Proton login)

4 Contacts (experimental)

docker compose -p user-{{.Name}} \ -f docker-compose.user.yml \ exec hydroxide \ hydroxide auth {{.Name}}@proton.me

After completing bridge login, the service status indicators on the main page will turn green as each bridge connects to Proton.
